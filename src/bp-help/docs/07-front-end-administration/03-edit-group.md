#Edit Group

Admins have access to quickly edit any group setting.

*   Edit Group Details
*   Edit Group Settings
*   Edit Group Photo
*   Edit Group Cover Image
*   Edit Group Members
*   Edit Group Requests
*   Delete Group - Deleting a group will delete all of the content it has created. It will be completely unrecoverable.
